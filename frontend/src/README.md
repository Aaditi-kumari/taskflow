# TaskFlow - Project Management Tool

## Live Demo
- Frontend: https://taskflow-nu-gules.vercel.app
- Backend API: https://taskflow-backend-xbl8.onrender.com

## GitHub Repository
https://github.com/Aaditi-kumari/taskflow

## Tech Stack
- Frontend: React.js
- Backend: Node.js + Express.js
- Database: PostgreSQL (Railway)
- Deployment: Vercel (Frontend) + Render (Backend)

## Features
- User Authentication (Login/Signup)
- Role-based Access (Admin/Member)
- Project Management
- Task Management with Priority & Status
- Team Management
- Activity Feed
- Notifications Dashboard

## Setup Instructions
1. Clone the repo: `git clone https://github.com/Aaditi-kumari/taskflow.git`
2. Backend: `cd backend && npm install && node server.js`
3. Frontend: `cd frontend && npm install && npm start`
